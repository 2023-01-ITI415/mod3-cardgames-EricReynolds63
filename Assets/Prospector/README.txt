	
	Pyramid Solitaire:
	1.) Create pyramid (1 at top, then 2, etc, 7 at bottom)
	2.) Rest of the deck goes at bottom
	3.) If possible, pair off pyramid cards to add to 13
		4.) If no pair possible flip top deck card
			5.)If possible, pair deck card with available cards
			6.)If no pair possible, deck card goes to waste pile
				7.) Waste pile card be combined with draw pile
					OR
				8.) Waste pile card can be paired with available pyramid card
		9.) Cannot pair hidden cards (hidden by row below)
	10.) Game ends when pyramid is cleared
		OR
	11.) No more moves can be made
